# Changelog

All notable changes to Travel Wizards will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Material 3 design system with dynamic color support
- Firebase Authentication (Google, Email/Password)
- Enhanced onboarding flow with travel preferences
- Multi-step trip planning wizard
- Trip details with itinerary, weather, packing lists
- Brainstorm AI-powered trip ideas
- Explore destinations and recommendations
- Settings with theme, language, and accessibility options
- Push notifications support
- Offline mode with local sync
- Responsive layouts for mobile, tablet, and web
- Multi-language support (10+ Indian languages)
- Test coverage for critical flows

### Fixed
- Removed TODO comments from codebase (per coding standards)
- Cleaned up non-standard documentation files
- Implemented search functionality in navigation
- Added destination and buddy management in trip planning
- Stabilized widget tests for onboarding and profile flows

### Security
- Secure Firebase configuration
- Environment-based secrets management
- No credentials in source code
